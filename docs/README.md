# GoNFC-Web | Documentation Bridge

Bu dizin, GoNFC projesinin web katmanına ait teknik ve sektörel dokümantasyonu içerir. 

## 🏗️ Proje Rolü (Redirect Bridge)
GoNFC-Web, kullanıcıların fiziksel NFC kartları ile dijital dünya arasında bağlantı kurmasını sağlayan bir **Yönlendirme Köprüsü** (Redirect Bridge) olarak tasarlanmıştır.

## 📂 Klasör Yapısı
- **/technical:** API referansları, Flutter mimarisi ve Firebase entegrasyon detayları.
- **/solutions:** NFC teknolojisinin endüstriyel kullanım senaryoları ve çözüm mimarileri.
